# Etapes d'installation de l'environnement
Afin de lancer l'environnement de travail, suivez ces étapes : 

1. Se rendre à la racine du projet et ouvrir la console de commande de votre choix
2. S'assurer d'avoir installé :
    - Docker (et démarré)
    - node.js
3. Lancer la commande : > docker compose up --build
